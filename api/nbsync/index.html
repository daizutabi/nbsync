    class Store(src_dirs: str | Path | Iterable[str | Path])

    Manage notebook files from one or more source directories.

    Provides a centralized interface for reading notebook files and caching their content for efficient access. Automatically reloads files when they have been modified on disk.

    Initialize a new Store instance.

    Attributes

    • src_dirs : list[Path] — List of source directories to search for notebook files.

    • nodes : dict[Path, NotebookNode] — Dictionary mapping file paths to their notebook nodes.

    • st_mtime : dict[Path, float] — Dictionary mapping file paths to their last modification times.

    • url : str — String representing the last accessed URL.

    Parameters

    • src_dirs : str | Path | Iterable[str | Path] — One or more directories to search for notebook files. Can be a single path or a collection of paths.

    Methods

    • find_path — Find the absolute path of a notebook file.

    • read — Read a notebook file and return its content.

    • write — Write a notebook node to a file.

    method Store.find_path(url: str) → Path

    Find the absolute path of a notebook file.

    Searches for the notebook file in the source directories. If the URL is an absolute path, it is returned directly.

    Parameters

    • url : str — The URL or relative path of the notebook file.

    Returns

    • Path — The absolute path to the notebook file.

    Raises

    • ValueError — If the file cannot be found in any source directory.

    method Store.read(url: str) → NotebookNode

    Read a notebook file and return its content.

    If the file has been modified since it was last read, it is reloaded. If no URL is provided, the last URL is used.

    Parameters

    • url : str — The URL or relative path of the notebook file.

    Returns

    • NotebookNode — The notebook content.

    method Store.write(url: str, notebook_node: NotebookNode) → None

    Write a notebook node to a file.

    Currently only supports writing to .ipynb files.

    Parameters

    • url : str — The URL or relative path of the notebook file.

    • notebook_node : NotebookNode — The notebook content to write.

    Raises

    • NotImplementedError — If the file format is not supported for writing.
